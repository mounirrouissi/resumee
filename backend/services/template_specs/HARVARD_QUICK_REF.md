## Key Measurements
- **Page**: US Letter (8.5" × 11")
- **Margins**: 0.75" all sides
- **Name**: 22pt Times-Bold, centered, underlined, black
- **Contact**: 10pt Times-Roman, centered
- **Sections**: 12pt Times-Bold, ALL CAPS, left-aligned
- **Body**: 10pt Times-Roman
- **Line spacing**: 1.2 (12pt leading for 10pt text)
- **Section spacing**: 8pt before, 6pt after
- **Bullet indent**: 20pt from left

## Content Rules

### Bullets (3-6 per role)
✓ Start with action verb (Led, Implemented, Designed)
✓ Include metrics (%, $, time, count)
✓ Keep to 2 lines max
✓ Tech stack in parentheses at end
✗ No "I" or personal pronouns
✗ No generic phrases ("responsible for")

### Dates
✓ Right-aligned
✓ Consistent format (2020 – 2024 OR Jan 2020 – Jun 2024)
✓ Use en-dash (–) not hyphen (-)

### Tense
✓ Present tense for current role
✓ Past tense for previous roles

## Section Order
1. Name + Contact (centered)
2. EDUCATION (most recent first)
3. EXPERIENCE (most recent first)
4. SKILLS (grouped by category)
5. PERSONAL INFORMATION (optional, 2-4 bullets)

## ATS Compliance
✓ Single column layout
✓ Standard section names
✓ Selectable text
✓ No images or graphics
✓ Times New Roman font
✓ Simple structure
✓ Keywords naturally placed

## File Output
- **Format**: PDF with embedded fonts
- **Filename**: `Lastname_Firstname_Role_Year.pdf`
- **Size**: 1 page preferred, 2 pages max
- **Quality**: High-quality, print-ready
